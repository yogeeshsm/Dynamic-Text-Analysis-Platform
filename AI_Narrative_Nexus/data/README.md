# Data Directory

Place your `cleaned_dataset.csv.csv` file in this directory.

The file should contain the Twitter US Airline Sentiment Dataset with at least the following columns:
- `text`: The tweet text
- `airline`: The airline name (optional)
- `airline_sentiment`: The sentiment label (optional)

All processed data files will be stored here:
- `clean_tweets.csv` - Preprocessed text data
- `sentiment_results.csv` - Sentiment analysis results
- `topic_results.csv` - Topic modeling results
- `topics.csv` - Topic summaries and keywords
